# blockup

Scaffold and serve bl.ocks.org gists. Compiles ES6 to ES5.

## usage

```sh
Usage: blockup <command>

Commands:
  new    scaffold and serve a new block
  serve  serve current block

Options:
  -h, --help  Show help                                       [boolean]

v0.0.3
```

## install

` npm install -g blockup`
